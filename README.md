# my-cnode
### 项目描述

使用React技术写自己的cnode社区

[体验地址](https://tian-cai.github.io/my-cnode/build/index.html#/)

### 技术栈

React + ES6 + React-Route4 + ant

### 感谢

感谢cnode社区提供的API

### 使用
```js
//克隆项目
git clone https://github.com/tian-cai/my-cnode
//安装
npm install
//本地运行
npm run start
//打包
npm run build

```
